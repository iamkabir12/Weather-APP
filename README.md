# 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/iamkabir12/Weather-APP.git
```

```bash
cd Weather-APP
```

### Install Packages

```bash
flutter pub get
```

### Add Your API Key

Replace

```dart
YOUR_API_KEY
```

with your own OpenWeatherMap API key.

### Run

```bash
flutter run
```
